# SPAN Digital BE Coding Test
Built with:
- Node.js: `20.18.0`
- NPM: `10.8.2`

## Setup
1. Run `npm install`.
2. Run `node index`.
3. You can write line per line the results or copy and paste the whole string (separated by line).
4. After you write all the results, press `Ctrl` + `D` to get the ranking.

## Test
Run `npm test`.
